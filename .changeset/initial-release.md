Initial release of react-kino and @kino/core.

react-kino provides cinematic scroll-driven storytelling components for React:
- `<Kino>` root provider
- `<Scene>` pinned scroll section
- `<Reveal>` scroll-triggered entrance animations
- `<Parallax>` speed-shifted layers
- `<Counter>` animated numbers
- `<CompareSlider>` before/after slider
- `<HorizontalScroll>` horizontal scroll sections
- `<Progress>` scroll indicator

@kino/core provides the framework-agnostic scroll engine with zero dependencies.
